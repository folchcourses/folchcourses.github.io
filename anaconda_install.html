Installing Continuum's Anaconda Python

Anaconda is a Python distribution that bundles the Python programming language and over 720 Python packages for scientific computing, some of which provide spatial analysis and/or GIS functionality.

  1. Go to the Continuum website and download the 64-bit version of Python 3.6 for your operating system.
  2. Install the downloaded file as you would any piece of software.
    • Use all the defaults
  3. Finish the install from the command line
    • On Mac open the Terminal application (Applications > Utilities > Terminal) on Windows open the Anaconda Prompt application (Start > Anaconda (64-bit) > Anaconda Prompt)
    • Update all the Python packages. Type (or copy-and-paste) the following and hit Enter: conda update --all

Install a Good Text Editor

A good text editor will make coding easier and more fun. The following are solid options, but others exist.
  • Mac: BBEdit
  • Windows: Notepad++
