Installing Continuum's Anaconda Python

Anaconda is a Python distribution that bundles the Python programming language and over 720 Python packages for scientific computing, some of which provide spatial analysis and/or GIS functionality.

  1. Go to the Continuum website and download the 64-bit version of Python 2.7 for your operating system.
  2. Install the downloaded file as you would any piece of software.
    • Use all the defaults, EXCEPT make Anaconda your default python
  3. Finish the install from the command line
    • On Mac open the Terminal application (Applications > Utilities > Terminal) on Windows open the Command Prompt application
    • Install Anaconda Navigator. Type (or copy-and-paste) the following and hit Enter: conda install anaconda-navigator
    • Upadte all the Python packages. Type (or copy-and-paste) the following and hit Enter: conda update --all
  4. The Navigator application can be found on Mac in Utilties > Navigator; on Windows it should be on your Desktop

Install a Good Text Editor

A good text editor will make coding easier and more fun. The following are solid options, but others exist.
  • Mac: TextWrangler
  • Windows: Notepad++

(OPTIONAL) Using ArcGIS with Canopy Python

When you install ArcGIS, you also get a version of Python, a limited set of open source packages and ESRI's python packages, called arcpy. This is a very basic setup. Anaconda Python will be cooked into ArcGIS at some point, but until that time we want a better option than what ESRI supplies.
