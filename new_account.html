Setting Up Github and Gitter Accounts for My Courses

  1. If you don't already have a Github account, create one.

  2. I encourage you to edit your GitHub profile to include a photo or some other image to distinguish you from the masses.

  3. If you don't already have a Gitter account, create one.
    • To see chats for the courses' private repositories: Log in to the website or app and click the "Create a Room" button in the left sidebar, then choose Repository. There should then be a link prompting you to add the private repository permission. Note: Don't actually create a new room. [Note: You might need to repeat this step a couple times.]
    • The Gitter folks ask for a bunch of permissions on your GitHub account. In my experience they have honored all the privacy settings and not messed up my repositories. Their long explanation of permissions.
  4. Email me your GitHub user name.

  5. Install related software
    • SourceTree. This software provides a nice interface to your GitHub repositories.
    • Gitter Apps. Gitter has desktop and mobile apps. Technically you can do everything on Gitter through their website so these apps are optional.
