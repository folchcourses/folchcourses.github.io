Viewing a Course Repository

The course repository is located in a private organizational account in the GitHub cloud. This means that it can only be viewed by invited GitHub users, i.e., students in the class. During the initial weeks of the course you will only be reading and "pulling" from the GitHub course repository. As we move into the project phase you will start to add content to your GitHub repository.

Github account

By now you should have done these things:

  1. Signed up for an account on Github
  2. Requested and received educational status with Github
  3. Sent me your Github user name
  4. Accepted my invitation to join the folchcourses Github organization
If you have not done all of these, follow the steps here.

Viewing the course repository online

In what follows we will be using the example of a student named Joe Johnson, with the Github user name of geonole. You should replace this with your GitHub user name in the steps that follow.

  1. Login to Github
  2. Click your user name in the upper-right corner. example
  3. Click the folchcourses icon in the lower-left corner. example
  4. Click the repository with your course name, semester and year. example
  5. Here you will find the syllabus, homework exercises and other course content. example

Cloning the course repository to your local hard drive

Cloning allows you to pull course content to your lab or personal computer. GitHub provides software to facilitate this process.
  • Mac
  • Windows
Follow these steps to get started with the GUI (only the first time you use it).
  1. Download the appropriate software for your computer, and install it.
  2. Startup the GUI, and follow the steps to link your GitHub account to the GUI.
  3. "Clone" the course repository to your local hard drive.
You can now pull (called "sync" in the software) course content and view it on your local machine.
