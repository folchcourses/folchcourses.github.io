Installing Enthought Canopy Python

Enthought Canopy is a Python distribution that bundles the Python programming language and over 200 Python packages for scientific computing, some of which provide spatial analysis and/or GIS functionality.

Get an Enthought Academic Account

  1. Go to the Enthought website
  2. Click Create Account in the upper-right. example
  3. Fill in the form using your fsu.edu email account, which will facilitate getting an academic account later.
  4. Once you have completed the form, Enthought will send an activation email. Follow the directions in this email.
  5. From the activation web page click the Request an academic license link at the bottom-left. example
  6. On the Academic web page, click the Request your Academic License button on the bottom-left. example

Downloading Enthought Canopy

  1. Login to the Enthought website
  2. Click the Downloads link on the left side. example
  3. The website will detect your operating system, and offer four different downloads: example
    • 32-bit Canopy Express
    • 32-bit Canopy Full ← choose this if you're on Windows AND you want Canopy to access ArcGIS's python packages; if you have an older computer you might need this.
    • 64-bit Canopy Express
    • 64-bit Canopy Full ← Mac and Linux users should choose this, along with Windows users not wanting to link to ArcGIS.
Note: Some installations of ArcGIS use 64-bit architecture and/or 64-bit Python code. If you have this version of ArcGIS you probably know it, and should install 64-bit Canopy Full.

Installing Enthought Canopy

  1. Double-click the file you downloaded and install it as you would any piece of software. Use all the defaults unless you're comfortable changing things.
  2. The software is now "installed", but is not really usable yet.
  3. Launch the Canopy application to begin the setup process. On a Mac you may need to go to the Applications folder and right-click the Canopy app and select Open (this is only required the first time).
  4. Unless you have a good reason not to, use the default location for the environment directory. example
  5. Unless you have a good reason not to, make Canopy your default Python. Click Start using Canopy. example
  6. Log in to your Enthought account in the Canopy application. example
  7. Click the Package Manager button. example
  8. Click Updates in the left pane. example
  9. Click the Install all Updates button in the lower-right. example
Congratulations, you now have a state of the art Python environment on your machine!

Playing nice with ArcGIS (optional; not needed for the course)

When you install ArcGIS, you also get a version of Python, a limited set of open source packages and ESRI's packages (e.g., arcpy). What you have done so far will not impact ArcGIS or the Python that comes with ArcGIS (at least as far as I've tested). The following steps allow you to access ArcGIS's python packages AND Canopy's large set of open source packages in the same Python environment. You can link Enthought Canopy and ArcGIS if:
  • You're on a Windows machine
  • You installed 32-bit Enthought Canopy (in most cases)
  • You have ArcGIS installed on your machine
  1. Figure out the path to the ArcGIS installation on your machine. It's probably on C:\ or some directory of programs on C:\. We will call this C:\path\to\ArcGIS\.
  2. Figure out the path to the Enthought installation on your machine. It's probably in your user space on C:\. We will call this C:\Users\your_name\AppData\Local\Enthought\. If you do not see the AppData directory on your machine, then it is probably hidden; follow this link to unhide it.
  3. Open a text editor (e.g., Notepad or Notepad++), and enter the following lines (you will replace \path\to\ with the path on your machine):
    4. # .pth file for arcpy
    C:\path\to\ArcGIS\Desktop10.1\bin
    C:\path\to\ArcGIS\Desktop10.1\arcpy
    C:\path\to\ArcGIS\Desktop10.1\ArcToolbox\Scripts
  4. Save the content of the text editor as zzArcGIS.pth.
  5. Move zzArcGIS.pth to C:\Users\your_name\AppData\Local\Enthought\Canopy32\User\lib\site-packages.
You should now be able to import arcpy into you Canopy Python environment. If you want more details on this process or want to import Canopy packages when you're in ArcGIS Python, click here.
